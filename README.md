# Quant Learning & Trading System

量化交易学习与实战系统

## 学习路径

本项目记录从零开始学习量化交易并开发完整量化系统的全过程。

### 阶段一：基础
- [ ] Python 编程基础
- [ ] Pandas/NumPy 数据处理
- [ ] 金融市场基础

### 阶段二：策略研究
- [ ] 技术指标与因子
- [ ] 经典策略（双均线、RSI、均值回归）
- [ ] 多因子模型

### 阶段三：回测系统
- [ ] Backtrader/Zipline 学习
- [ ] 回测框架搭建
- [ ] 绩效分析

### 阶段四：实盘系统
- [ ] 券商 API 对接（QMT/Ptrade）
- [ ] 交易信号执行
- [ ] 风险控制

### 阶段五：系统集成
- [ ] 数据 pipeline
- [ ] 策略调度
- [ ] 监控告警

## 项目结构

```
docs/          - 学习笔记与文档
```

```bash

git clone git@github.com:weijie-lee/quant.git

# 安装依赖
pip install -r requirements.txt
```
